LIMINAL

LoRa Radio-Space Sensing · Open Source · MIT

Turn 4 Heltec V2 LoRa nodes into a real-time radio topology sensor. Instead of sending messages through space, LIMINAL reads the space itself.

Connect a Heltec WiFi LoRa 32 V2 via USB-C. Requires Chrome or Edge.

First time? Hold BOOT button while plugging in USB. If not detected, install CP210x drivers.

Frequency: Defaults to 868 MHz (EU/UK). For US/AU, clone repo and change FREQUENCY to 915E6.

Simulation — cursor = human body in radio field.

CURSOR = HUMAN BODY IN RADIO SPACE
Click "Connect Serial" and select your Heltec V2 port.

HARDWARE

  • 4 x Heltec WiFi LoRa 32 V2
  • USB-C cables (1 min for bridge)
  • 3 x USB power banks
  • LoRa antennas

STEP 1 — FLASH

Use the Flash Firmware tab, or build from source:

pio run -e node0 -t upload

STEP 2 — PLACE NODES

Corners of sensing area, 3-15m apart, waist height.

STEP 3 — RUN BRIDGE

pip install pyserial websockets
python bridge/bridge.py

STEP 4 — VISUALISE

Open visualiser/index.html in Chrome.

CONCEPT

4 nodes ping each other via LoRa TDMA. RSSI/SNR for all 6 pairs maps how radio energy flows through space and is disrupted by bodies.
